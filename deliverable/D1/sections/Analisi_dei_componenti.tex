\section{Analisi dei componenti}
\subsection{Definizione dei componenti}
Di seguito viene riportata, sulla base dei requisiti funzionali precedentemente descritti, la definizione dei vari componenti del nostro sistema.

\CMP{Gestione accesso e autenticazione}\label{cmp:accesso_autenticazion}
Il componente si occupa di gestire l'accesso degli utenti al sistema e di verificarne l'identità tramite un sistema di identità digitale. 
\paragraph{Interfacce richieste:}
\begin{itemize}
    \item \textbf{Richiesta di login:} richiesta da parte dell'utente di poter essere reindirizzato all'identity provider scelto per fare l'accesso.
    \item \textbf{Autenticazione dall'identity provider:} restituzione dei dati identificativi dell'utente da parte dell'identity provider scelto per l'accesso.
    \item \textbf{Conferma di esistenza nel database:} conferma da parte del database dell'esistenza dell'utente nel database.
\end{itemize}
\paragraph{Interfacce fornite:}
\begin{itemize}
    \item \textbf{Richiesta all'identity provider:} reindirizzamento dell'utente verso il sistema di autenticazione digitale scelto dall'utente.
    \item \textbf{Verifica di esistenza nel database:} richiesta al sistema di gestione del database di verificare l'esistenza dei dati forniti nel database.
    \item \textbf{Autenticazione:} notifica al resto del sistema riguardo i privilegi dell'utente.
    \item \textbf{Reindirizzamento dashboard:} reindirizzamento alla dashboard personale dell'utente qualora risultasse già registrato.
    \item \textbf{Richiesta di creazione profilo:} reindirizzamento al flusso di creazione del profilo qualora l'utente non risultasse registrato.
\end{itemize}

\CMP{Gestione creazione profilo}\label{cmp:creazione_profilo}
Il componente si occupa di gestire la creazione di un nuovo profilo da parte di un utente.
\paragraph{Interfacce richieste:}
\begin{itemize}
    \item \textbf{Richiesta di creazione profilo:} richiesta di creazione di un nuovo profilo (proveniente da CMP\ref{cmp:accesso_autenticazion}) dopo un login concluso con il non riconoscimento dell'utente nel database.
    \item \textbf{Recapito mail:} inserimento da parte dell'utente di un recapito mail.
\end{itemize}
\paragraph{Interfacce fornite:}
\begin{itemize}
    \item \textbf{Registrazione utente:} invio dei dati dell'utente al database per registrarne il profilo in caso di buona riuscita della procedura di creazione.
\end{itemize}
